# Guardian AI

Guardian AI is a comprehensive security and monitoring platform designed to protect high-trust enterprises and critical infrastructure.

## Project Structure

- `guardian-landing/`: The main landing page and frontend application built with React, Vite, and Tailwind CSS.
- `src/`: Source code for the landing page.
- `public/`: Static assets including brand logos.

## Getting Started

To run the landing page locally:

1. Navigate to the `guardian-landing` directory:
   ```bash
   cd guardian-landing
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## Production Build

To create a production build:
```bash
npm run build
```

The output will be in the `dist/` directory.
